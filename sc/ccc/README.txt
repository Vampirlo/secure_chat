Открывать все файлы необходимо БЛОКНОТОМ.

Файлы, которые необходимо настроить:
1. keyfile - 128 битный ключ. Диапазон 00 - FF. Пример - 01 04 02 03 01 03 04 0A 09 0B 07 0F 0F 06 03 00
2. serverip.dat - ip сервера для подключения клиента. Пример - 100.126.255.205

Файлы, которые трогать не надо:
1. message.aes - хранит последние зашифрованные данные;
2. in.dat - хранит последние данные, которые были зашифрованы;
3. out.dat - хранит последние данные, которые были расшифрованы;

Остальные файлы вас ебать не должны.


Алгоритм использования:
(ОБЯЗАТЕЛЬНО ПУТЬ ДОЛЖЕН БЫТЬ БЕЗ ПРОБЕЛОВ(и без других любых символов))
1. прописать ключ шифрования и указать ip для подключения (для клиента);
2. запустить secure_chat.exe;
3. Выбрать ориентацию;
4. написать "своё" имя;
5. когда захотели написать сообщение - нажали Escape (esc);
6. пишем любое сообщение, которое не нарушает законодательство Российской Федерации (пример сообщения - "Го_трахнем_Лизу_Самсонову")
7. нажимает Enter;
